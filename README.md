# 🎯 Mini JavaScript Projects

This repo contains a collection of small projects I built while learning and practicing **HTML, CSS, and JavaScript**. Each project helped me understand core concepts by actually building something useful and interactive.

## 🚀 Projects Included

### ⏱️ Stopwatch

A simple stopwatch with start, stop, and reset functionality. Helped me understand timing functions like `setInterval`.

### 🧮 Calculator

A basic calculator that performs arithmetic operations. Built to practice DOM manipulation and event handling.

### 📝 To-do List

A task manager where you can add, delete, and manage tasks. Focused on working with arrays, local storage, and dynamic UI updates.

### ✊ Rock-Paper-Scissors

A fun game against the computer. Helped me practice logic building, condition handling, and randomization.

## 💡 Why I Built This

I created these projects to apply what I learned and improve my hands-on skills. Instead of just watching tutorials, I focused on building real mini-projects to strengthen my understanding.

## 🛠️ Tech Used

* HTML
* CSS
* JavaScript

## ▶️ How to Run

1. Clone the repository
2. Open the project folder
3. Run `index.html` in your browser
